# 🎭 Hidden Motive（疑云）

AI剧本杀 - 基于Dify的智能推理游戏

## 🔗 在线体验
**https://caichewu.github.io/Hidden-Motive-/**

## 🎮 功能特色

### 3个剧本
- 🏔️ **疑云**：暴风雨夜，山顶别墅谋杀案
- 🚂 **消失的乘客**：列车密室毒杀案  
- 📱 **直播间**：网红直播时离奇死亡

### AI角色
- 每个角色有独特语音风格（语速、音调不同）
- 林医生：冷静缓慢 | 苏记者：锐利带刺
- 周司机：紧张结巴 | 陈厨师：憨厚朴实

### 交互方式
- 文字输入提问
- AI语音+文字同时回复
- 支持多轮推理、剧本切换

## 🛠️ 技术栈
- **Dify**：LLM工作流编排
- **GitHub Pages**：前端部署
- **Web Speech API**：语音合成

## 📖 使用方法

1. 打开网页：https://caichewu.github.io/Hidden-Motive-/
2. 在代码第20行填写你的Dify API Key
3. 选择剧本，输入"问[角色名]：[问题]"
4. 例如："问林医生：你昨晚在哪里？"
